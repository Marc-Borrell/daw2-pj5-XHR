## Test

If you want to store data inside mongoose Docker, you need to stop your local MongoDB if it's running
because if you don't stop the service node will try to connecto to yor local mongoDB database instead 
the mongoDB that's inside the Docker. <br>

You can run this command in Windows as admin to stop the service of mongoDB: net stop MongoDB
